*GUI x CLI
---------------
Navegação

Windows
- tab : autocompletar
- cd : change directore
- dir : listar
- mkdir : cria um diretorio
- del/rmdir: del, deleta arquivos. rmdir, deleta o diretorio
- cls: limpar a tela
- > : redirecionador de fluxo

Unix
- cd : change directore
- ls : listar
- mkdir : cria um diretorio
- rm -rf
- clear: limpar a tela = ctrl + L
